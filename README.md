# Album
=======
# 微信小程序——炫酷相册


## 功能介绍
    基于TensorFlow2.1，图像识别方向的应用：用户登录/注册上传图片，后台进行识别返回图片标签，前端自动分类；选择你要生成相册的图片生成你想要的相册样式。

## 注意
    这只是本应用的前端，后台待上传

## 目录结构
```c
    ┠--  start        // 登录
    ┠--  index        // 图片选择/上传页面
    ┠--  select       // 图片分类页面
    ┠--  myalbum      // 相册生成页面


```
